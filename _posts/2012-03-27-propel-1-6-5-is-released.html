Propel 1.6.5 is now available, it's the second bug fixes release but it also contains a gift. Thanks to Denis Dalmais, and Cedric Lombardot, two awesome contributors, Propel now has an EXPLAIN feature, and sfPropelORMPlugin, and PropelBundle already use it.

[[posterous-content:gIGnAqfgCitvbrvGtqag]]

[[posterous-content:xHfvmvwmtJgfywFzjmkf]]

The release is available on GitHub under the 1.6.5 tag, as PEAR package (runtime & generator) and as archives (ZIP and TAR or by browsing files.propelorm.org too).

This release is also available through Composer: propel/propel1, and now uses the classmap autoloading feature provided by Composer. If you are using it to manage your dependencies, you should use this feature ;)

As usual, the API documentation is at: http://api.propelorm.org/, and here is the CHANGELOG. Thanks to all contributors.
